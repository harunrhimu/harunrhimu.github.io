## Power BI Developer | Microsoft Fabric Enthusiast | Data Storyteller

Hi, I'm **Harun** — a self-taught **Power BI Developer** from Bangladesh with a background in Population Sciences and a deep passion for turning raw data into compelling visual stories.

I specialize in building **enterprise-grade dashboards**, designing **semantic models**, and architecting **end-to-end analytics solutions** using the Microsoft data stack. With Microsoft Fabric, I'm exploring the future of unified analytics — from data ingestion to AI-powered insights, all in one platform.

### What I Do

**Power BI & Fabric**
- Design interactive, KPI-driven dashboards and paginated reports
- Build optimized semantic models with DAX & data modeling best practices
- Implement Row-Level Security (RLS) and workspace governance
- Create Dataflows Gen2 and data pipelines in Microsoft Fabric
- Work with Lakehouses, Warehouses, and Notebooks in Fabric
- Automate report distribution and data refresh schedules

**Data Engineering & Analysis**
- Clean, transform, and automate data pipelines (Python, SQL, Power Query)
- Conduct exploratory data analysis and trend forecasting
- Translate complex datasets into business intelligence and strategic insights
